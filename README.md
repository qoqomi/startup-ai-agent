## 주제

> **AI 스타트업 투자 가능성 평가**
>
> - 국내외 AI 스타트업을 조사하고

> - 투자 가능한 스타트업에 대한 평가 보고서를 작성하는 Agentic RAG 설계 및 구현

## 평가 기준표를 만들기위한 문서 조사

사용문서: startup-ai-agent/documents/startup_doc.pdf

- **Berkus Method** (Dave Berkus, 1990s-2020)
  - 출처: Angel Capital Association 공식 문서
  - 근거: 20년 실전 투자 검증
  - 핵심: "천 개 중 한 개도 재무 예측 안 맞음"
- **Scorecard Method** (Bill Payne, 2001-2019)
  - 출처: Frontier Angels / ACA
  - 근거: Wiltbank 2007 연구 (10개 중 1-2개만 대박)
  - 핵심: 지역/산업 맥락 반영
- **Startup = Growth** (Paul Graham, 2012)
  - 출처: Y Combinator 창립자 에세이
  - 근거: 수학적 증명 (주 5% → 4년 후 12.6배)
  - 핵심: 성장이 스타트업 정의
- **PMF Theory** (Marc Andreessen, 2007)
  - 출처: Andreessen Horowitz 공동창립자 블로그
  - 근거: Rachleff's Law (Benchmark Capital 경험)
  - 핵심: Market > Team > Product
- **Default Alive/Dead** (Paul Graham, 2015)
  - 출처: Paul Graham Essays
  - 근거: "과도한 채용 = 스타트업 사망 1순위"
  - 핵심: 생존 가능성 즉시 계산

### 선정 기준

- “스타트업”이 가진 특색을 고려하여 설계
- 각 5개의 문헌이 상호 보완이 가능함
  - 실전 검증 : 수천건의 실제 투자에서 검증됨
  - 시간 검증 : 10~30년 살아남은 방법론을 제시함 → 실제 필드에서 존재하는 사람의 평가기준을 바탕으로 투자 결정을 내림
  - 업계 표준: 실리콘밸리 투자자들이 실제로 사용함
  - 상호 보완: 각자 다른 위험을 평가함
  - 측정 가능 : 주관이 아닌 객관적임

### 3-1. 위험 감소 평가

**용도**: 매출 전(Pre-revenue) 초기 단계 / 빠른 스크리닝

| 평가 항목          | 배점           | 평가 기준                                    | 점수          |
| ------------------ | -------------- | -------------------------------------------- | ------------- |
| **아이디어 품질**  | 0~50만불       | 5년 내 $20M 매출 가능성? 문제-솔루션 적합성? |               |
| **프로토타입**     | 0~50만불       | 작동하는 시제품 존재? 기술 검증 완료?        |               |
| **경영진**         | 0~50만불       | 업계 경험? 실행력? 코칭 가능성?              |               |
